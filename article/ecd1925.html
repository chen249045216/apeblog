快速搭建漂亮的个人博客

  • vscode
  • markdown
  • github
  • hexo

    为什么需要一个独立的博客？

  1. 本地markdown管理文章文件，真正将自己的文章把我在自己的手中，随意可以导出到任何平台上
  2. 没有广告
  3. 有自己的域名
  4. 有丰富的主题可以选择

开始搭建

安装git

  • Windows: git.
  • Mac: 用 Homebrew, MacPorts 或 installer安装.
  • Linux (Ubuntu, Debian): sudo apt-get install git-core
  • Linux (Fedora, Red Hat, CentOS): sudo yum install git-core

安装node.js

  • Windows: Install it with nvs (recommended) or nvm.
  • Mac: 用 Homebrew, MacPorts 或 installer安装.
  • Linux (DEB/RPM-based): Install it with NodeSource.
  • 其他: Install it through respective package manager. Refer to the guide provided by Node.js.

安装hexo

npm install hexo-cli -g

初始化（空目录）

hexo init

查看效果

hexo s

github

  1. 新建repository
  2. 设置settings->pages
  3. 添加CNAME文件
  4. 配置主题
  5. https://mengd.js.org/article/Quick-start.html
  6. 安装git插件

    npm install hexo-deployer-git –save

  7. 生成html

    hexo g

  8. 部署页面

    hexo d

可能会遇到的问题

配置github ssh-keygen

  1. ssh-keygen -t rsa -b 4096 -C “邮箱地址@qq.com”

    注意：自定义保存地址选择 /Users/用户名/.ssh/github_rsa

  2. 修改~/.ssh 中的config文件
  3. vim ～/.ssh/config
  4. ssh-add -K ~/.ssh/github_rsa
  5. pbcopy < ~/.ssh/github_rsa.pub

链接优化

npm install hexo-abbrlink –save
